Newton-Raphson Method Documentation

The Newton-Raphson method uses the formula:

xn+1 = xn - f(xn)/f'(xn)

It requires both the function and its derivative. The method converges quadratically when near a root.

Bisection Method Documentation

The Bisection method uses the intermediate value theorem:

c = (a + b)/2

It requires an interval [a,b] where f(a) and f(b) have opposite signs. The method converges linearly.

Secant Method Documentation

The Secant method approximates the derivative:

xn+1 = xn - f(xn) * (xn - xn-1)/(f(xn) - f(xn-1))

It requires two initial guesses but doesn't need the derivative. Convergence is superlinear.

Regula Falsi Method Documentation

The Regula Falsi method uses a weighted approach:

c = (a*f(b) - b*f(a))/(f(b) - f(a))

Like bisection, it requires an interval where f(a)f(b) < 0. It typically converges faster than bisection.

Fixed-Point Iteration Documentation

Fixed-point iteration solves g(x) = x:

xn+1 = g(xn)

Convergence depends on |g'(x)| < 1 near the root. You must provide a suitable g(x) function.
